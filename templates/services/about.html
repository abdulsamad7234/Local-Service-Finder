About this project

This is a simple Django project built to help users find and share local services. It also demonstrates how backend and frontend parts connect in a real application.

What I implemented

  • Django models, views, templates, and admin panel usage
  • Category-based service listings with proper relationships
  • Search, filtering, form validation, and pagination

Admin usage

Services can be added and approved from the Django admin panel. Only approved services are shown on the website.
